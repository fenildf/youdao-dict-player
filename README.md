# youdao-dict-player
自动播放有道单词本中的单词（主要是读音和搜图）

## 环境要求

Node 环境

## 安装方法

克隆本仓库：

```
git clone https://github.com/lewis617/youdao-dict-player.git
```

进入项目根目录：

```
cd youdao-dict-player
```

安装依赖

```
npm i
```

设置账户

```
export YOUDAO_ACCOUNT="username;password"
```

链接到全局

```
npm link
```

## 使用方法

在命令行中任意目录下输入：

```
ydp
```
